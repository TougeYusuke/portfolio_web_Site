# リサイクルショップ「Portfolio Recycle (Sample)」

## プロジェクト概要
架空のリサイクルショップのランディングページ（LP）です。
「信頼感」と「親しみやすさ」をテーマに、和風モダンなデザインで構築しました。
Web制作スキルを示すためのポートフォリオ作品です。

## 技術スタック
- **HTML5**: セマンティックなマークアップ
- **CSS3**: Vanilla CSS, レスポンシブデザイン（Media Queries）, Flexbox/Grid
- **JavaScript**: スムーススクロール, ハンバーガーメニュー制御

## 主な機能・特徴
1.  **レスポンシブ対応**
    - スマートフォン（375px～）、タブレット、PCのすべてのデバイスで最適に表示されます。
2.  **モバイルメニューUI**
    - 右上に固定されたハンバーガーメニュー。
    - 展開時に画面全体を覆うオーバーレイデザイン。
3.  **デザイン**
    - 高級感のあるネイビーとゴールドを基調としたカラーパレット。
    - 信頼性を高めるすっきりとしたレイアウト。

## ファイル構成
```
project-root/
├── index.html       # メインページ構造
├── style.css        # デザイン・スタイル定義
├── script.js        # 動的機能（メニュー、スクロール）
└── images/          # 画像アセット
```

## 閲覧方法
`index.html` を各種Webブラウザ（Chrome, Edge, Safari等）で開いてください。
ビルドツールは不要で、そのまま動作します。

## 注意事項
このサイトはポートフォリオ用の**サンプル**であり、実在する店舗ではありません。
記載されている電話番号や住所はダミーデータです。
